
# 🪙 Heads or Tails

Heads or tails? Now it's easy to decide! And simple.
## 📋 Requirements

• tkinter

• pillow
  
## ⚙️ Installation

• Clone the project;

```bash
  git clone https://github.com/NotYello/Heads-or-Tails
```

• Go to the project directory;

```bash
  cd Heads-or-Tails
```

• Install the required packages;

```bash
  pip3 install pillow
```

• Run the Python file;

```bash
  python3 main.py
```
